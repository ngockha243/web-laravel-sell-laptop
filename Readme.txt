*Trang web chính:

B1.   copy toàn bộ Folder "BLTWeb" vào phần thư mục htdocs của Xampp.
B2: Khởi động Xampp (Apache, MySQL) với port 8888 (Apache) (Dạ tùy vào port ở trên máy   cài đặt)
B3:   sử dụng trình duyệt Web với đường link: http://localhost:8888/phpmyadmin
 - Tại đây, giao diện phpMyAdmin xuất hiện.   chọn New => Import => Chọn tệp => elaravel.sql => Go.
B4:   sử dụng trình duyệt Web với đường link: http://localhost:8888/BTLWeb
B5: Test sản phẩm.
